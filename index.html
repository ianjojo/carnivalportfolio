Colour Machine

A simple but helpful tool for picking colour schemes for a project. Created with Javascript and using the Color API, this app provides you with various colour schemes and allows you to copy the HEX codes by clicking upon a colour name. Click a colour to make it your new seed colour and a new scheme will be generated. Follow the colour!
