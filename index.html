Face Detector

An experiment with machine learning and tensorflow.js, using a pre-trained object detection ML model called COCO-SSD. Accessing the user's webcam and then determining if the object displayed is a face and, if not, its' best guess. While the machine is smart, it is not a genius.

Will the computer machine detect your face?!

The Diabolical

Colour Machine

A simple but helpful tool for picking colour schemes for your next project. Using the Color API, this app provides you with various colour schemes and allows you to copy the HEX codes by clicking upon a colour name. Click a colour to make it your new seed colour and a new scheme will be generated. Follow your heart!

What shade does your heart desire?

Ghostly Forest

An experiment with p5.js and L-systems - creating lifelike trees using recursion and pi to draw branches, then initiating several rows of instances of the trees and redrawing everything with generative variables everytime the page is clicked.

You'll never see the same forest twice!

The Mummified

Music Box

A nod to the great Brian Eno. Using p5.js alongside tone.js to play random sets of notes within a scale at random intervals. Each time you reload the page, a new iteration of music will be heard, never the same twice. Moving your mouse to the right side of the screen adds reverb, while moving back to the left mixes it down. I am fascinated by the potential for generative music via code.

What ghastly notes will you hear today?
