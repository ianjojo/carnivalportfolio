Face Detector

Will the computer machine detect your face?!

My first time using tensorflow.js. Accessing the user's webcam and then determining, with tensorflow, if the object displayed is a face. Also works with many other objects and gets confused by many more. Written in HTML/Javascript.

The Diabolical

Colour Machine

A simple but helpful tool for picking colour schemes for a project. Created with Javascript and using the Color API, this app provides you with various colour schemes and allows you to copy the HEX codes by clicking upon a colour name. Click a colour to make it your new seed colour and a new scheme will be generated. Follow the colour!

Ghostly Forest

You'll never see the same forest twice!

My first successfuly experiment with p5.js - creating lifelike trees using recursion and pi to draw branches, then initiating several rows of instances of the trees and redrawing everything with generative variables everytime the page is clicked.

the zombified

music generator

Using p5.js alongside tone.js to play random sets of notes at random intervals. Moving your mouse to the right side of the screen adds reverb, while moving back to the left mixes it down. I am very fascinated with generative music and was inspired fpr this project by mr Brian Eno.
